# Real-Time Streaming Analytics Platform

This project demonstrates a **high-throughput streaming platform** using **Apache Kafka**, **Spark Streaming**, and **AWS Kinesis**. It enables real-time ingestion and processing of event data for operational dashboards and analytics.

## Features
- Real-time data ingestion with Kafka and Kinesis
- Spark Streaming for processing high-volume events
- Scalable and reliable streaming workflows
- Integration with dashboards and analytics tools

## Project Structure
- `src/`: Core scripts for Spark, Kafka, and Kinesis streaming
- `notebooks/`: Demo notebook for streaming data processing
- `data/`: Sample event data
- `config/`: Configuration files
- `scripts/`: Scripts to run Kafka and Kinesis streaming pipelines
